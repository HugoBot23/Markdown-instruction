Чтобы добавить ненумированные списки, необходимо пункты выделить звездочкой (*).
Например, вот та:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимые пункты просто пронумировать.
Например, вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
